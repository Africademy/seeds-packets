Get the Code

For SCSS or Javascript:

  • yarn add @sproutsocial/seeds-typography
  • npm install @sproutsocial/seeds-typography --save

Notes

  • Proxima Nova is our font.
    Sass
    font-family: $Typography-family;
    Javascript
    { style: { fontFamily: TYPOGRAPHY_FAMILY } }
  • Only two weights are permitted: regular and semibold, which is called bold for simpler naming
    Sass
    Regular
    font-weight: $Typography-weight--regular;
    Bold
    font-weight: $Typography-weight--bold;
    Javascript
    Regular
    { style: { fontWeight: TYPOGRAPHY_WEIGHT_REGULAR } }
    Bold
    { style: { fontWeight: TYPOGRAPHY_WEIGHT_BOLD } }
  • You can use rems if you want to by setting $Typography-unit: rem; in your Sass file. Use the $Typography-size--base variable to set the base size used to calculate the rem value.

Sizes

Line heights are calculated to ensure at least three lines align to an 8pt grid. Use the associated line-height values only.
